feat(bridge): add Fordefi bridge implementation

This submission introduces Substrato 1066, which is the FORDEFI-CATHEDRAL-BRIDGE. It establishes the bridge implementation for the external MPC wallet platform Fordefi, connecting it as an Interface Layer to the existing Cathedral ontology. The substrate links heavily to Substrato 1042.4 (Liquidity-Integrity-Bridge), 954 (Axiarquia), and 989.z.4 (ZK-Circom). The integration includes a Python implementation for connecting with Fordefi's API, unit tests for validation, detailed README mapping to the ecosystem lore, and an updated substrate_registry.yaml reflecting the new 1066 node.
